# OrderManagement

![Order Management system](https://i.imgur.com/u8Ikm3O.jpeg)

## Features

- 側邊欄開合
- 串接資料庫
- 依照訂單付款狀態分類查看
- 搜尋訂單編號
- 編輯訂單資料
- 刪除訂單
- 限制訂單顯示數量及換頁
